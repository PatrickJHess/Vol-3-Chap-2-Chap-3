# Function Imported by Chapter Three.

Chapter Threes notebooks import the helper functions:

[validate_date](https://patrickjhess.github.io/Imported-Functions/validate_date.html#validate-date-helper-function). This function checks for date or datetime objects and returns date object. 

[scheduled_pay_dates](https://patrickjhess.github.io/Imported-Functions/scheduled_pay_dates.html#scheduled-pay-dates). This function calculates scheduled payment dates of bonds. 

[adjust_bond_pay_dates](https://patrickjhess.github.io/Imported-Functions/scheduled_pay_dates.html#scheduled-pay-dates). Calculates actual payment dates and amounts from scheduled dates. 


These functins are part of a module downloaded from DropBox. These and other functions within the [module](https://patrickjhess.github.io/Imported-Functions/intro.html#functions-imported-by-the-volume-basic-concepts-of-fixed-income) are intended to shift focus from coding to financial concepts.
